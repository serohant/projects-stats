# PHP Project Stats

This is simple Project analyzer

you can easily get 
- Start Time
- End Time
- Total Hours Spend 
- Total Lines
- Lines Per Day

# Install

You can install with

    gh repo clone serohant/projects-stats

## Usage
just open in browser
